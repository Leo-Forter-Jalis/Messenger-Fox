## Описание проекта
### На данный момент существует 4 модуля: 
>> MessFox  
> | Client-Desktop  
> | Protocol  
> | Server    
> | Shared

## Protocol: 
### Его задачей является определить, какие запросы и ответы могут быть у нашего мессенджера. Формат там в основном DTO, но есть и Enum'ы и класс с static полями.
## Shared:
### Тут в основном утилитарные классы(шифрование паролей, обертка над логированием, работа со временем, кодер/декодер Json и работа с файлами) которые могут использоваться у server и client частях.
